# RESTAURANT PAGE

A simple restaurant page built using Vanilla Javascript to generate its entire content.

![screenshot](images/restaurant.png)


## Built With
- JavaScript
- Webpack
- HTML & CSS

## Getting Started
To get a copy up and running follow these simple example steps:-

- Clone https://github.com/Godswilly/restaurant-page.git to your local machine.
- cd into the restaurant-page directory and navigate into the dist directory.


## Usage
- Clone the project to your local machine
- Run npm install
- npm run watch
